# Front Matter仕様

## 基本形式

すべてのマークダウンファイルの先頭にYAML形式のFront Matterを記述します。

```markdown
---
title: ページタイトル
description: ページの説明文（省略可）
---

# ページタイトル

本文...
```

## プロパティ一覧

| プロパティ | 必須 | 説明 |
|-----------|------|------|
| `title` | 推奨 | ページタイトル。サイドバー・ブラウザタイトルに使用 |
| `description` | 任意 | ページの説明。メタタグ・OGPに使用 |
| `modtime` | 任意 | 更新日時。Sitemap.xmlのlastmodに使用 |

## titleの決定ロジック

以下の優先順位でタイトルが決定されます：

1. Front Matterの`title`
2. マークダウン内の最初の`# 見出し`
3. ファイル名（プレフィックス除去後）

```markdown
<!-- 1. Front Matterのtitle -->
---
title: インストールガイド
---

<!-- 2. 最初のH1見出し -->
# インストールガイド

<!-- 3. ファイル名: 01_installation.md → "installation" -->
```

## descriptionの決定ロジック

1. Front Matterの`description`
2. 本文の最初の段落（150文字まで）

## modtimeの形式

```yaml
modtime: "2024-01-15 10:30:00"
```

- 形式: `YYYY-MM-DD HH:MM:SS`
- 省略時: ファイルの更新日時が使用される

## タイトルのみのindex.md

グループ名だけを設定し、リンクを作らない場合：

```markdown
---
title: グループ名
---
```

本文を書かないと、サイドバーではタイトルが表示されるがリンクにはなりません。

## 出力テンプレート

新規ページを作成する際は以下の形式を使用してください：

```markdown
---
title: ページタイトル
description: このページの概要を1-2文で
---

# ページタイトル

導入文...

## セクション1

内容...

## セクション2

内容...
```
